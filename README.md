## 📌 Project Overview
Cyber Lab is an AI-powered cyber security assistant designed to help users identify potential threats such as malicious links, suspicious files, and unsafe online activities.  

The platform focuses on **simplicity, speed, and accessibility**, making cyber security tools usable for everyone—even non-technical users.

---

## 🚀 Features
- 🔍 Analyze suspicious URLs and inputs  
- 🤖 AI-based threat detection system  
- ⚡ Fast and real-time response  
- 🎯 Clean and user-friendly interface  
- 📱 Fully responsive design  

---

- **shadcn-ui**
